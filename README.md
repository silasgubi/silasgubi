# Silas Gubitoso

Sao Paulo, Brazil. I build home automation systems that work when the internet does not.

My Home Assistant setup does things visitors actually notice: light color temperature
that shifts with the sun position throughout the day, and retro-style dashboards
running on old iPads mounted around the house. Under the hood: autonomous irrigation
with fail-safe fallbacks and UPS monitoring with automatic alerts. Most of it is not
public yet - still removing hardcoded keys and writing real docs.

Currently learning Meshtastic for off-grid monitoring in rural areas where
cellular coverage is unreliable.
